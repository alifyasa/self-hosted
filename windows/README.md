# Windows 11 Clean Setup

What to do for a fresh install of Windows 11:

## [Windows Debloat](https://github.com/Raphire/Win11Debloat)

[Full Instruction](https://github.com/Raphire/Win11Debloat?tab=readme-ov-file#advanced-method)

Flags:

```
-RemoveDevApps -ClearStart -DisableTelemetry -DisableBing -DisableSuggestions -DisableLockscreenTips -RevertContextMenu -ShowHiddenFolders -ShowKnownFileExt -TaskbarAlignLeft -DisableCopilot -DisableWidgets -HideChat
```
